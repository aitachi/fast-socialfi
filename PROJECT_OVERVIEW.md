## 项目简介

Fast SocialFi 是一个去中心化社交金融平台，整合社交网络与DeFi机制。

### 核心功能

**社交功能**:
- 用户注册/登录 (钱包签名)
- 内容创作与分享
- 点赞/评论/关注
- 实时通知系统

**DeFi功能**:
- ERC20治理代币
- ERC721内容NFT
- 质押挖矿系统
- DAO治理

### 技术栈

- **区块链**: Solidity 0.8.20, Hardhat, OpenZeppelin
- **后端**: Node.js, TypeScript, Express
- **数据库**: PostgreSQL, Redis
- **消息队列**: Kafka
- **搜索**: Elasticsearch
